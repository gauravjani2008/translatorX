About TranslateX

Version1.0.0
Format supportMXLIFF 1.2, XLIFF 1.2
Tag typesg, x, bx, ex, ph, mrk, sub, it (all XLIFF inline)
BackendNone — fully client-side
APIGoogle Gemini (generativelanguage.googleapis.com)
ℹ️ How tag preservation works: All Memsource/XLIFF inline tags are replaced with ⟦TAG1⟧ style placeholders before sending to Gemini. After receiving translations, placeholders are swapped back with the original XML tags. The Gemini prompt strictly instructs the model never to modify these placeholders.
⚠️ Anti-hallucination: Temperature is set to 0.1. The prompt uses explicit zero-hallucination directives. Output is validated against source segment count and tag presence. Always perform a human review before using translations in production.
